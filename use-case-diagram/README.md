# Use Case Diagram for Airbnb Clone Backend

## Actors

- **Guest**
- **Host**
- **Admin**

## Interactions

### Guest
- Register account
- Login
- Manage profile
- Search properties
- Book property
- Cancel booking
- Make payment
- Review property
- Receive notifications

### Host
- Register account
- Login
- Manage profile
- List property
- Edit property
- Delete property
- View bookings
- Respond to reviews
- Receive notifications

### Admin
- Login
- Monitor users
- Manage listings
- Oversee bookings
- Monitor payments
